## Ambiente de Desenvolvimento

Clone o repositório:

```bash
$ cd ~/your/directory/
$ git clone https://github.com/FranciscoHeronildo/corpsystem-front.git
$ cd corpsystem-front
```

Instale as dependencias:

```bash
$ yarn install
```

Inicie a aplicação:

```bash
$ yarn dev
```

Acesse a aplicaçã na porta _5173_:

```
http://localhost:5173/
```
